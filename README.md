
# 饮料自动售卖机管理系统

涉及工具：

编辑器：```VScode```；前端开发：```html，css，JavaScript```；后端开发：```php，mySQL```

## 系统需求

本系统需满足以下需求：

1. 管理员通过计算机管理分布在学校各处的售卖机，调整饮品结构，增加饮品库存，适应师生需求。

2. 管理员对师生的需求情况能做好全面掌握，及时获取历史销售记录。

3. 管理员及时进行补货。

4. 对饮品的销售管理。
